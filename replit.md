# InsideOut Brand.ng

## Overview
A React-based personalized greeting cards and story builder application. Built with Vite, React, Tailwind CSS, and various UI components from Radix UI and MUI.

## Project Structure
- `src/` - Main source code
  - `app/` - Application components and pages
    - `components/` - Reusable UI components
    - `pages/` - Page components (HomePage, AvatarCreationPage, StorySubmissionPage, ConfirmationPage)
  - `styles/` - CSS stylesheets
  - `main.tsx` - Application entry point
- `index.html` - HTML entry point
- `vite.config.ts` - Vite configuration
- `package.json` - Dependencies and scripts

## Tech Stack
- React 18.3.1
- Vite 6.3.5
- Tailwind CSS 4.x
- React Router DOM 7.x
- Radix UI components
- MUI components

## Development
The frontend runs on port 5000 with Vite dev server.

### Scripts
- `npm run dev` - Start development server
- `npm run build` - Build for production
